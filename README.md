#  Flutter商城app
> ***让一部分人先看见未来***

为了练手和挑战，我利用空闲时间，用Flutter写了一个仿淘宝和小米的App

由于快速开发，且个人也没有太多时间去测试，所以大家运行时如发现问题可以提到Issues，我看看是Flutter的问题还是代码的问题。

## 开源不易，麻烦给个Star吧！我会根据大家的关注度和个人时间持续更新代码！
